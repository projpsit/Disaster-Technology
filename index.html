แรดคือ?

ชื่อสัตว์เลี้ยงลูกด้วยน้ำนมขนาดใหญ่ในวงศ์ Rhinocerotidae เป็นสัตว์กีบคี่ มีนิ้ว 3 นิ้ว หนังหนา ขาสั้น ตาเล็ก หูตั้ง ประสาทการมองเห็นไม่ค่อยดีแต่ประสาทการดมกลิ่นดีมาก ชอบนอนแช่ปลักโคลนตามหนองน้ำ ในประเทศไทย มี 2 ชนิด คือ แรดนอเดียว เรียกทั่วไปว่า แรด หรือแรดชวา ( Rhinoceros sondaicus ) และกระซู่ หรือแรดสุมาตรา ( Dicerorhinus sumatrensis ) มีนอ 2 นอ.
